🔹 HR Analytics Dashboard (Power BI)

This project is an interactive HR Analytics Dashboard built using Power BI to analyze employee data and uncover key insights related to workforce trends, attrition, and performance.
________________________________________
📌 🔍 Objective

The goal of this project is to:

•	Analyze employee data 

•	Identify attrition patterns 

•	Understand workforce distribution 

•	Help HR teams make data-driven decisions 

________________________________________

🛠️ Tools & Technologies Used

•	Power BI 

•	Excel / CSV Dataset 

•	Data Cleaning & Transformation (Power Query)


•	DAX (Data Analysis Expressions) 

________________________________________
📊 Dashboard Features

•	Total Employees Overview 

•	Attrition Rate Analysis 

•	Department-wise Employee Distribution 

•	Gender & Age Group Analysis 

•	Salary Slab Insights 

•	Job Role Breakdown 

•	Education Field Analysis 

________________________________________
📈 Key Insights

Here are the important insights from the dashboard:

1️⃣ Attrition Analysis

•	Higher attrition observed in low salary slab employees 

•	Most employees leaving are from early career stage 

2️⃣ Department Insights

•	Maximum employees belong to Sales & R&D 

•	Attrition is higher in Sales department 

3️⃣ Age Group Trends

•	Employees aged 25–35 show highest turnover 

•	Senior employees have lower attrition 

4️⃣ Gender Distribution

•	Workforce is moderately balanced 

•	Slight variation in attrition between male and female employees 


5️⃣ Job Role Insights

•	Roles like Sales Executive & Laboratory Technician show higher attrition

•	Manager-level roles are more stable
